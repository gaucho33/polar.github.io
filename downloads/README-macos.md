# Polar Desktop Application - macOS

This is a placeholder for the Polar Desktop Application for macOS.

## Installation Instructions

1. Download the polar-macos.dmg file
2. Open the DMG file
3. Drag Polar.app to your Applications folder
4. Launch Polar from Applications
5. Enter your license key when prompted

## System Requirements

- macOS 10.15 (Catalina) or later
- 4GB RAM minimum
- 500MB disk space
