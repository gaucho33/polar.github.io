# Polar Desktop Application - Linux

This is a placeholder for the Polar Desktop Application for Linux.

## Installation Instructions

1. Download the polar-linux.AppImage file
2. Make it executable: `chmod +x polar-linux.AppImage`
3. Run the application: `./polar-linux.AppImage`
4. Enter your license key when prompted

## System Requirements

- Ubuntu 20.04 or later (or equivalent)
- 4GB RAM minimum
- 500MB disk space
